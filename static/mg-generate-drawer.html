<mg-generate-drawer> — standalone verification

Isolates the shared Generate form (full-parity build: 6 image + 3 video + 1 audio ref, negative prompt, Channel, full model roster) from both hosts, so a component bug and a mount-bridge bug can't be confused. Serve the gallery, then open this at /static/mg-generate-drawer.html on the same machine (it calls the localhost-gated /api/price / /api/loom/generate / /api/upload). Image and video slot picks are serviced by <mg-gallery-picker> via the mg-pick-request event, filtered by its kind hint — exactly how a real host wires it. Audio refs upload directly (no picker involved, matching how the component itself works).
