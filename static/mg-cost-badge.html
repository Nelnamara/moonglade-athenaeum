<mg-cost-badge> — standalone verification

Isolates the shared cost/free-card badge from both hosts. The component never fetches — the host feeds it a parsed /api/price response — so every state below is driven by a button, and no state here can spend anything. The last card does call the real, read-only /api/price (serve the gallery, open this at /static/mg-cost-badge.html on the same machine) to prove the live response shape classifies correctly.
